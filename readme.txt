
CORP-E-BANK est une simulation du système bancaire
il s'agit d'un laboratoire d'experimentation des concepts liés à la fois, au langage python 
mais aussi des concepts propres à l'environnement bancaire.

cette application n'est pas executé dans un virtualenv pour cela vous dévez :
1- créer le virtualenv avant de clôner le projet
2- puis installer flask dans ce virtualenv
3- ensuite copier ces fichier dans votre virtual env
